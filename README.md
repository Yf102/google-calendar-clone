# Google Calendar Clone

React + TypeScript + Vite + Docker + Nginx

## First time
```bash
npm i
```

## Run a development server
```bash
npm run dev
```

## Run a production build

```bash
npm build
npm run preview
```

## Docker

Only first time
```bash
docker-compose up --build --no-recreate -d
```
From the second time onwards
```bash
docker-compose up -d
```
